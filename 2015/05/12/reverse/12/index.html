逆向基础第十二篇：控制流程 之 for循环

By John·Huang

发表于 2015-05-12

文章目录

for 循环是三种循环（do，while，for）中最复杂的一种。for循环由赋初值、设置循环条件、设置循环步长 这三条语句组成。在程序中for被使用的频率是最高的。

for循环测试代码：

int _tmain(int argc, _TCHAR* argv[])
{
    int nSum = 0;
    for (int i = 0;i < argc ; i++)
    {
        nSum += i;
    }
    printf("%d \r\n",nSum);
    return 0;
}

反汇编代码如下：

通过上图，我们看出，赋初值只执行一次。然后不断自增、执行代码块，往上跳。通过循环流程，我们可以总结其特性。

总结：

Step1：赋初值，跳转到循环判定部分，然后执行代码块

Step2：代码块末尾，向上跳，跳转到自增（修改步长部分，这里是自增，视实际情况而定）
Step3：循环判定部分，然后执行代码块，向上跳跳到Step2
