Requirements

必须安装ImageMatic才能对图片进行处理

If you’re on Mac OS X, you’ll want to run the following with Homebrew:

brew install imagemagick

检查是否安装成功：

执行 which convert 理论上来说应该返回 /usr/local/bin/convert 说明ImageMatic安装成功了。

编辑 config/environments/development.rb 配置command_path

Paperclip.options[:command_path] = "/usr/local/bin/"

安装

编辑gemfile

gem "paperclip", "~> 4.2"

然后

bundle install

使用

model

class User < ActiveRecord::Base
  has_attached_file :avatar, :styles => { :medium => "300x300>", :thumb => "100x100>" }, :default_url => "/images/:style/missing.png"
  validates_attachment_content_type :avatar, :content_type => /\Aimage\/.*\Z/
end

migrateion

创建一个migration

rails g migration add_avatar_columns_to_users

编辑 xxx_add_avatar_columns_to_users

class AddAvatarColumnsToUsers < ActiveRecord::Migration
  def self.up
    add_attachment :users, :avatar
  end

  def self.down
    remove_attachment :users, :avatar
  end
end

new view

<%= form_for @user, :url => users_path, :html => { :multipart => true } do |form| %>
 <%= form.file_field :avatar %>
<% end %>

controller

def create
  @user = User.create( user_params )
end

private

# Use strong_parameters for attribute whitelisting
# Be sure to update your create() and update() controller methods.

def user_params
  params.require(:user).permit(:avatar)
end

show view

<%= image_tag @user.avatar.url %>
<%= image_tag @user.avatar.url(:medium) %>
<%= image_tag @user.avatar.url(:thumb) %>

删除

把属性设置为nil，然后保存即可

@user.avatar = nil
@user.save
