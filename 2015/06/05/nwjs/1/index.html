使用nwjs创建跨平台的本地html5应用

nwjs简介

nwjs你可能不熟悉，但是提到node-webkit应该很多人会听说过，其实他们是一个东西。具体的我就不多介绍了，可以到官网了解详情。

官网:nwjs.io

github: https://github.com/nwjs/nw.js

环境：MacOSx

第一个nwjs应用

我下载的是64位的Mac版：http://dl.nwjs.io/v0.12.2/nwjs-v0.12.2-osx-x64.zip ，解压之后将nwjs.app拖到 /Applications 目录下。

编辑 .bash_profile ,添加一个别名

# alias to nw
alias nw="/Applications/nwjs.app/Contents/MacOS/nwjs"

刷新一下bash的环境变量

source ~/.bash_profile

到这里nwjs算是配置完成了。

现在我们可以创建自己的应用了，首先创建一个文件夹，名字为myapp。

在myapp文件夹下创建index.html 

<!DOCTYPE html>
<html>
  <head>
    <title>Hello World!</title>
  </head>
  <body>
    <h1>Hello World!</h1>
    We are using node.js <script>document.write(process.version)</script>.
  </body>
</html>

在myapp文件夹下创建package.json

{
  "name": "nw-demo",
  "version": "0.0.1",
  "main": "index.html"
}

OK，现在准备工作已经完毕，进入myapp目录，在终端中运行：

nw .
